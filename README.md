# Gym Management System

Sistema de gerenciamento de academia desenvolvido em Java, com foco em Programação Orientada a Objetos.

## Objetivo

Este projeto foi criado para simular um sistema simples de academia, permitindo cadastrar alunos, planos, pagamentos, treinos e gerar relatórios.

O projeto une conhecimentos de tecnologia com experiência prática na área fitness e personal training.

## Funcionalidades

### Alunos

- Cadastrar aluno
- Editar aluno
- Excluir aluno
- Listar alunos
- Buscar aluno por CPF

### Planos

- Plano Bronze
- Plano Prata
- Plano Ouro
- Plano Premium

Cada plano possui:

- Nome
- Valor
- Duração
- Benefícios

### Pagamentos

- Registrar pagamento
- Verificar mensalidade em atraso
- Histórico de pagamentos

### Treinos

- Cadastrar treino do aluno
- Adicionar exercícios
- Informar séries e repetições

Treino A
Supino - 4x10
Crucifixo - 3x12
